awesome WM and VirtualBox Windows

Date Di 31 Mai 2022 By Basti Category Blog. Tags awesome_wm VirtualBox

Since i'm using the really awesome awesome wm tiling window manager, the behaviour of the VirtualBox has anoyed me.

Everytime my put machine to suspend, the windows of running vms were moved to my main display.

But I finally found a that the following configuration option in the awesome configuration file:

client.disconnect_signal("request::geometry", awful.ewmh.client_geometry_requests)

Will prevend this behaviour. No more searching for openend vm windows - Yippiee
