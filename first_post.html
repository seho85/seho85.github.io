First Blog Post

Date Di 31 Mai 2022 By Basti Category Blog.

Finally after a few years of thinking about creating a blog, I finally did it.

I set up my blog on which I'll report about technical stuff that encountered to me in my daily work.

I'll use Pelican to generate my blog. And will be completly static, so no commenting is possible for now.

Due my latest project (i'll report about that later) I finally came in touch with python and thats one of the reason I've choosen Pelican to create my blog.

For now i'll use an existing pelican-theme that suited best for my needs. Maybe i'll create my own when I find some time and when i'm interessted in doing some CSS stuff.
