Arch Linux on Raspberry PI4

Date Do 23 Juni 2022 By Basti Category Blog. Tags Raspberry Pi ArchLinux

I was able to buy a new Raspberry PI 4. And because I like ArchLinux, which is also running on my notebook and desktop machine, I decided to use ArchLinux on the Raspberry PI 4.

There are many installation guides that describe the installation, so I don't want to repeat them here.

Basically I followed this instructions but with a small tweak.

Instead of using the aarch64 64bit edition of ArchLinux I use the the 32bit armv7 edition.

So instead of

wget http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-aarch64-latest.tar.gz
bsdtar -xpf ArchLinuxARM-rpi-aarch64-latest.tar.gz -C root

I used

wget http://hu.mirror.archlinuxarm.org/os/ArchLinuxARM-rpi-armv7-latest.tar.gz
bsdtar -xpf ArchLinuxARM-rpi-armv7-latest.tar.gz -C root
